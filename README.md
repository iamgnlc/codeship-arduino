# codeship-arduino

Will poll last build status from Codeship and fire the relative LED on arduino
board.

Rename `config.example.js` to `config.js` and add your codeship details to it.

### Run

```
$ yarn install
$ yarn start
```

### Exit

Press `[CTRL] + [C]`.
